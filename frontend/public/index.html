Categories

Bot Buster streamlines content to respective APIs across different categories. Three main categories come implemented with the base product to detect AI-generated content and malicious bot behaviour.

  • AI Generated Content: Determines if text is AI generated, each API returns a continuous score from 0 to 100 of how probable it is AI generated.
  • Misinformation: Establishes if text is misinformation, featuring Bot Buster's own in-house misinformation model that returns a contiuous score from 0 to 100, of the probability it is misinformation.
  • Hate Speech: Classifies text as hate speech or not, where each API provides a discrete score of 0 being not flagged or 100 as flagged.

Additionally, users are permitted to upload APIs of their choosing to exisiting or new categories through the Add API page found on the sidebar.

Generate Results

Upon submitting text, analyse insights from all selected APIS across the categories. Results are summarised with graphical visualisations to conclude the nature of the text at a glance and display in-depth scores based on each request which highlights the sentences responsible for the score.

Text is chunked into multiple requests and sent to the APIs, where the responses will be summarised and flagged based on predefined thresholds for categories worth investigating.

  • Unflagged (0-19%): The results score is low enough for it not to be of interest.
  • Potentially Flagged (20-79%): When the score may need to be investigated based on the user's discretion, where the score is not too low nor too high.
  • Flagged (80-100%): The results score is extremely high and it can be safely assumed that it is flagged.
